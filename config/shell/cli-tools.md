# CLI Toolkit Reference

| Alias / Command      | Tool            | Description                              |
| --------------------- | --------------- | ---------------------------------------- |
| `ls`, `l`, `ll`, `la`, `lt`, `tree`, `llm`, `l.` | eza | Modern ls with icons/git |
| `cat`                 | bat             | Syntax-highlighted file viewer           |
| `rm`                  | trash           | Safe delete to trash                     |
| `cd` / `z`            | zoxide          | Smart cd, learns frequent dirs           |
| `grep` → `rg`        | ripgrep         | Fast recursive search                    |
| `find` → `fd`        | fd              | Fast file finder                         |
| `git diff`            | delta           | Beautiful diffs, side-by-side            |
| `man` → `tldr`       | tlrc            | Community cheat sheets                   |
| `du` → `dust`        | dust            | Visual disk usage                        |
| `df` → `duf`         | duf             | Visual disk free                         |
| `jq`                  | jq              | JSON processor                           |
| `hyperfine "cmd"`    | hyperfine       | Benchmark commands                       |
| `lg`                  | lazygit         | Terminal git UI                          |
| `ld`                  | lazydocker      | Terminal docker UI                       |
| `lwt`                 | lazyworktree    | Git worktree TUI                         |
| `btm`                 | bottom          | System monitor                           |
| `gh`                  | gh              | GitHub CLI — PRs, issues, repos          |
| `fzf`                 | fzf             | Fuzzy finder — powers Ctrl+R, Ctrl+T, Tab |
| `starship`            | starship        | Prompt theme                             |
| `micro`               | micro           | Terminal text editor                     |
| `glow`                | glow            | Render markdown in terminal              |
| `repomix`             | repomix         | Pack repo into single file for LLMs      |
| `onefetch`            | onefetch        | Git repo stats summary                   |
| `yt-dlp`              | yt-dlp          | Download videos from YouTube etc.        |
| `ncdu`                | ncdu            | Interactive disk usage explorer           |
| `fastfetch`           | fastfetch       | System info display                      |

## Keybindings

| Key         | Action                      |
| ----------- | --------------------------- |
| `Ctrl+R`    | Fuzzy history search (fzf)  |
| `Ctrl+T`    | Fuzzy file finder (fzf)     |
| `Alt+C`     | Fuzzy cd (fzf)              |
| `Tab`       | fzf-powered completion      |
| `Esc Esc`   | Prepend sudo                |
| `Up`/`Down` | History substring search    |
